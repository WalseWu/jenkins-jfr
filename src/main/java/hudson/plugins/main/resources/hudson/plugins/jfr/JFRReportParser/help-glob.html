Specify the path pattern to the JRockit Flight Recording report files, relative to the workspace root.

  • You can specify one pattern here for directorys or files.
  • You could use an Ant 'fileset' pattern.
  • Files has same pattern will be merged in one graph, e.g. if the pattern like **/a.b.c.Alltests-*.jfr, then a.b.c.Alltests-node1.jfr and a.b.c.Alltests-node2.jfr will be merged.

Default Values are:

  • **/*.jfr, which means all jfr files will be merged in one graph.
