Specify the events and event values of JRockit Flight Recording you want to show on the JFR Report Graphs, the format should be like: <event path>':'<field-id>[,<field-id>];. For multiple events, please using ";" to seperate them, E.g.: os/cpu_load:user,system,total; vm/prof/memory_usage:total:init, total:used. Here, "total:init" and "total:used" are two event fields of the event vm/prof/memory_usage.

  • You can specify multiple files and directories separated by semicolon.

Default Values are:

  • os/cpu_load:user,system,total; vm/prof/memory_usage:total:init, total:used;
