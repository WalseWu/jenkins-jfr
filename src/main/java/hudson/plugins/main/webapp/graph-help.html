Specify the title and size (width and height in pixel) for JFR Report Graphs generated by this JFR files pattern.

  • Title: Each JFR Report Pattern will generate one or more graphs. If there are more than one JFR files merged into the graph, this TITLE value will be show as its title, if there is only one JFR file in the graph,the title will show as the JFR file name.
  • Width: The Width in pixel of the Graph. The default value is 600 if you leave it empty.
  • Height: The height in pixel of the Graph.The default value is 400 if you leave it empty.
